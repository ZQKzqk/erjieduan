nrm
  
  是一个基于 node 环境的 工具
   作用：专门用来能管理 npm 这个工具的下载地址的工具
  
       我可以使用 nrm 这个工具来切换 npm 的下载地址

       说明： 以后我们管 “下载地址” 这个名词 镜像元地           址


nrm  的安装
   因为 nrm 是一个基于 node 环境的工具
    
   我们可以使用 npm 来 安装
 
   nrm 是安装在电脑操作系统里的 只要安装一次 就行了

   npm i -g nrm   表示安装一个全局的nrm

        之后就可以随时使用nrm 去管理 npm的下载地址

nrm 的卸载
  
   npm un -g nrm


nrm检测

   输入指令 nrm --version


常用指令
   nrm test  
        测试网络环境 会出现npm 的可用镜像元地址的网络延         迟
   
        

   nrm use 
       
       切换镜像元地址  切换后 下载地址就回改变
