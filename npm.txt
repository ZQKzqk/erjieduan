npm 
   是一个 基于  node 环境  的包管理器

   如果想执行npm 这个软件  电脑里需要有 node 环境

 包管理器
   
   管理:上传 下载

   包 ：第三方内容
   
   一个专门在命令行进行  包 下载 上传  的工具

 npm 检测 
   
     打开命令行  npm -v

    
  
下载包  
    
    输入 npm i 包名

    下载指定版本 npm i  报名@版本号


初始化项目


   输入指令 npm  init 
             或 npm init-y
    
          一直 确定初始化 出现一个package.json

              的文件 里面记录着你所有下载过的包
   


查看一个包的可用版本
            npm view 包名 versions


卸载包
     方式1 
         打开node-modules 文件夹手动删除
     方式2
         npm un 包名

清楚缓存
   
   
  清楚网络缓存
    打开命令行   目录无所谓
       npm cache clean -f
   清楚本地缓存
     => 打开我的电脑
          => ca
          => users
          => 你的名字
          => appdata(这个文件夹可能是隐藏文件夹)
          => roaming
          => 删除里面的 npm-cache 文件件

     
npm 在什么情况下会有缓存
        + 你下载包的时候, 因为网路原因, 下载到一个临界点          的时候, 网络断开了
        + 就有可能会出现下载缓存
        + 缓存分成两种
          1. 网络缓存
          2. 本地缓存
        + 导致一个结果
          => 你以后每次下载都会失败    

查看包以前有过的版本
       npm view 包名 versions
